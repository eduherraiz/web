Desactivar el altavoz (pc speaker) en linux
Escrito por Edu Herraiz on 4 de Octubre de 2012 en Sin categoría.

Última actualización en 27 de Diciembre de 2014.

Crea un script en  /etc/profile.d/matar_altavoz.sh con el siguiente contenido:

setterm -blength 0
set bell-style none
amixer set 'PC Speaker' 0% mute
amixer set 'PC Beep' 0% mute
amixer set 'Beep' 0% mute

Dale permisos de ejecución:
chmod +x /etc/profile.d/matar_altavoz.sh

Y ejecútalo:
/etc/profile.d/matar_altavoz.sh
