Script con beep para hacer sonar la marcha imperial
Escrito por Edu Herraiz on 16 de Diciembre de 2011 en Linux, Programación.

Última actualización en 17 de Abril de 2015.

Me acordé de un script antiguo que pululaba por un foro de estudiantes de informática de la UIB. Lo encontré y lo publico aquí para quede para la posteridad.
Podéis ver como define las notas en variable y mediante el comando beep ejecuta las notas con un pequeño retraso configurable entre ellas. Hace tiempo lo hicimos sonar en el cluster de nuestro CPD, lástima que no conserve algún vídeo.
 
#!/bin/bash
MIb=”312.1″
FA1=”370.0″
SOL=”392.0″
SIb=”466.2″
RE=”587.33″
MI8b=”622.254″
DELAY=50
NEGRA=”300″
PUNTET=$(( NEGRA*3/4 ))
MITJ=$(( NEGRA/4 ))
beep -f $SOL -l $NEGRA -D $DELAY -r 3
beep -f $MIb -l $PUNTET -D $DELAY
beep -f $SIb -l $MITJ -D $DELAY
beep -f $SOL -l $NEGRA -D $DELAY
beep -f $MIb -l $PUNTET -D $DELAY
beep -f $SIb -l $MITJ -D $DELAY
beep -f $SOL -l 600 -D $DELAY
beep -f $RE -l $NEGRA -D $DELAY -r 3
beep -f $MI8b -l $PUNTET -D $DELAY
beep -f $SIb -l $MITJ -D $DELAY
beep -f $FA1 -l $NEGRA -D $DELAY
beep -f $MIb -l $PUNTET -D $DELAY
beep -f $SIb -l $MITJ -D $DELAY
beep -f $SOL -l 600 -D $DELAY
 
Edit: El vídeo que comenta worm es de la propia Aulalinux de la UIB y ejecutando el script en todos los ordenador a la vez.
